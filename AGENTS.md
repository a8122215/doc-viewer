# AGENTS.md

## このプロジェクトの目的

このプロジェクトは、同じ `Projects/` フォルダ配下にある各プロジェクトの `docs/**/*.md` を読み取り専用でHTML表示する個人用ビューアです。

## 実装方針

- 読み取り専用を維持する。
- 任意パス読み込みAPIを作らない。
- `Projects/<project>/docs/**/*.md` 以外を読まない。
- Markdownを原本とする。
- HTMLは表示用の生成物として扱う。
- Mermaidは `securityLevel: "strict"` を使う。
- サーバーは `127.0.0.1` 限定で起動する。
- 通常起動では `0.0.0.0` でlistenしない。
- Docker起動では、ホスト側ポートを `127.0.0.1` に固定し、`DOC_VIEWER_ALLOW_CONTAINER_HOST=true` を設定した場合に限りコンテナ内の `0.0.0.0` listen を許可する。

## 禁止事項

- ブラウザからのファイル編集機能を追加しない。
- ファイル削除機能を追加しない。
- ファイル移動・リネーム機能を追加しない。
- 任意の絶対パスを読めるAPIを追加しない。
- `docs` 以外のソースコードや秘密ファイルを表示対象にしない。
- Docker の `../:/projects` マウントを書き込み可能にしない。
